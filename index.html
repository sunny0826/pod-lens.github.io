Kubectl Pod Lens

Show pod related resources

One-Cmd Installation

kubectl krew install pod-lens

Kubectl Plugin

kubectl-pod-lens is a kubectl plugin that show pod-related resource information.

Display Pod-Related Resource

  • Workloads(Deployment,StatefulSet,DaemonSet)
  • Namespace
  • Node
  • Service
  • Ingress
  • ConfigMap
  • Secret
  • PVC/PV
  • HPA
